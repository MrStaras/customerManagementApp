# ConsultPro

A lightweight Client Management System built with Python and Flask.

## Features

* **Roles:** Administrators (Full Access) and Consultants (View Only).
* **Management:** Track Clients, Projects, and Tasks.
* **Dashboard:** Visual analytics and status overviews.
* **Security:** Secure login and registration.

## Tech Stack

* Python & Flask
* SQLite
* HTML / CSS / JavaScript

## How to Run

**1. Install Dependencies**
Run this command in your terminal:
`pip install Flask Flask-Login Flask-SQLAlchemy python-dotenv`

**2. Start the Application**
Run this command:
`flask --app cms run`

**3. Open in Browser**
Go to: `http://127.0.0.1:5000/`
